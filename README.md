# 5312_FinalProject
ISG 5312 Final Project
This is a reanalysis of the ball python genome assembly published in 2025. The original study is under bioproject PRJNA1217506 on NCBI. The goal of the original study was to construct a novel genome assembly for the species Python regius (Ball python). Pythons are a good model to study extreme metabolic and organ structural plasticity due the unique nature of significant organ remodeling, metabolic alterations, and associated changes in gene expression involved in their feeding cycle. In previous years, the close cousins Python bivittatus (burmese python) were used commonly as models since a genome assembly for this species was produced in 2013. However, this assembly was highly fragmented and with the Burmese python serving as an invasive species in some areas, using this model regularly becomes increasingly difficult. With the Ball python being a much more common species among the pet trade, it became apparent how valuble a good quality genome assembly for this species would be.

This reanalysis is an attempt to broadly reproduce the genome assembly. The documentation in the original study listed most of the computational tools that were used, but none with specific reference code. As a result, the reanalysis differs slightly in some tools used and how they were used.
