# 5312_FinalProject

This is a reanalysis of the ball python genome assembly published in 2025. The original study is under bioproject PRJNA1217506 on NCBI. The goal of the original study was to construct a novel genome assembly for the species Python regius (Ball python). Pythons are a good model to study extreme metabolic and organ structural plasticity due the unique nature of significant organ remodeling, metabolic alterations, and associated changes in gene expression involved in their feeding cycle. 

The aim of this reanalysis will be to broadly reproduce the genome assembly using publicly available sequencing data and standard bioinformatics tools provided by the University of Connecticut HPC. While the original study reported the tools used, it did not include fully reproducible code or pipelines. As a result, this project reconstructs the workflow using commonly accepted methods in genome assembly and polishing, with an emphasis on transparency and reproducibility. Although some differences in tool choice and parameterization exist, the overall structure of the pipeline closely follows the original study’s approach.

The workflow begins with the acquisition of raw sequencing data from NCBI, including both Illumina paired-end reads and Oxford Nanopore long reads. Illumina reads are first processed to remove adapters and low-quality sequences. The genome is then assembled using long-read data, which provides the structural backbone of the assembly. This initial assembly is subsequently polished in multiple stages: first using long-read polishing to correct systematic errors, and then using short-read polishing to improve base-level accuracy.

Following assembly and polishing, both Illumina and Nanopore reads are aligned back to the genome to generate coverage information. These alignments are used in a contamination assessment step, where tools are applied to evaluate sequence origin based on coverage and taxonomic classification. Contigs that do not meet quality or length thresholds are removed to produce a filtered assembly.

The final assembly is evaluated using standard quality assessment tools. Genome completeness is assessed using conserved single-copy orthologs, while general assembly statistics such as contig length and N50 are calculated to provide a broader view of assembly quality. Together, these metrics provide a comprehensive assessment of the reconstructed genome.
